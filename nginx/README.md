# NGINX

=========

## Install

### Step 0

```
sudo add-apt-repository ppa:nginx/stable -y
```

### Step 1

```
sudo apt update && sudo apt install nginx -y
```

Done!

## Update

```
sudo apt update && sudo apt upgrade -y
```

## Uninstall

```
sudo apt purge nginx -y
```
