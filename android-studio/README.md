# Android Studio

=========================

## Install

### Step 0

Installing Java.

```
sudo apt install openjdk-11-jdk
```

### Step 1

Install Android Studio through Snap store

```
sudo snap install android-studio --classic
```

Done!

## Update

```
sudo snap refresh
```

## Uninstall

```
sudo apt purge openjdk-11-jdk -y
sudo snap remove android-studio
sudo apt autoremove -y
```
